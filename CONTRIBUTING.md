# 🤝 Guía de contribución

¡Gracias por querer contribuir a ParafraseAI!

## Cómo contribuir

1. **Fork** el repositorio
2. **Clona** tu fork: `git clone https://github.com/TU_USUARIO/Parafraseo.git`
3. **Crea una rama**: `git checkout -b feature/nombre-de-tu-funcion`
4. **Haz tus cambios** y commitea: `git commit -m "Añade: descripción clara"`
5. **Push**: `git push origin feature/nombre-de-tu-funcion`
6. **Abre un Pull Request**

## Estilo de código

- Usa nombres descriptivos en inglés para variables y funciones
- Comenta el código en español para mantener consistencia
- Respeta la estructura existente
- Prueba tus cambios antes de enviar el PR
